# 経営ダッシュボード

試算表PDFデータを自動解析し、経営指標をビジュアル化するStreamlitダッシュボードです。

## 機能

- 📊 KPI自動集計（売上・粗利・営業利益）
- 📈 月別トレンドグラフ
- 🔍 科目別詳細分析
- 👔 AIによるエグゼクティブ・サマリー

## 使い方

1. `input_data/` フォルダにPDF試算表を配置
2. `streamlit run src/app.py` で起動
3. ブラウザで http://localhost:8501 にアクセス

## デプロイ

Streamlit Community Cloud でホスティング可能です。
